# Obstrategy

観測戦略を作成するためのツール。

## インストール

1. このリポジトリをクローンします。

    ```bash
    git clone https://github.com/your_username/Obstrategy.git
    ```
    
## 使い方

1. `Obstrategy`ディレクトリに移動します。

    ```bash
    cd Obstrategy
    ```

2. 以下のコマンドを実行して、観測戦略を作成します。

    ```bash
    python make_strategy.py
    ```
